Omni-Router

Multi-provider AI chat assistant with smart routing, quota management, debate mode, and a code agent — all running locally as an Electron desktop app.

View on GitHub

Install in one line

Prerequisites: Node.js 18+, npm, git

$ curl -fsSL https://raw.githubusercontent.com/mamoonk/omni-router/master/install.sh | bash
PS> iwr -Uri https://raw.githubusercontent.com/mamoonk/omni-router/master/install.ps1 | iex
$ git clone https://github.com/mamoonk/omni-router.git
$ cd omni-router
$ npm install
$ npm run build
$ npm run dev

Features

🧠

Smart Routing

Four routing strategies decide where each message goes:

  • Smart — Scores providers by intent fitness, quota, speed, and failure history
  • Cheapest — Prioritizes lower-tier providers to minimize cost
  • Fastest — Prioritizes low-latency providers
  • Round Robin — Cycles evenly through all available providers
🔌

20+ Providers

Gemini, OpenAI, Anthropic (Claude), Groq, Mistral, Cohere, DeepSeek, Together AI, Fireworks AI, OpenRouter, NVIDIA NIM, Perplexity, xAI Grok, Eden AI, SiliconFlow, DashScope (Qwen), AI21 Labs, Cerebras, HuggingFace, Cloudflare, SambaNova, DeepInfra, Stability AI, Replicate, Fal.ai, Pollinations.ai and more.

🎯

Intent Classification

Messages are automatically classified into one of five intents before routing:

  • Factual — Questions about facts, definitions, explanations
  • Code — Programming tasks, function writing, debugging
  • Long Document — Messages over 4000 characters
  • Creative — Brainstorming, writing, open-ended tasks
  • Image — Image generation requests
🛡️

Provider Failover & Degradation

If a provider returns a 429 (rate-limited) or other error, the router marks it degraded with a configurable cooldown, falls through to the next-best provider, and retries via exponential backoff (30s, 60s, 120s, max 3 attempts).

💬

Debate Mode

Two AI models discuss and refine each other's answers:

  • Primary (answerer) generates an initial response
  • Critic (reviewer) identifies errors, gaps, and improvements
  • Refinement round produces the final answer
🤖

Code Agent

Reads and modifies source code directly — works on the app itself or any external project. Scans the codebase, keyword-matches relevant files, injects them as context, and writes changes via structured edit blocks.

📊

Quota Management

Per-provider rate limits enforced in-memory with SQLite-backed tracking:

  • RPM — Requests Per Minute
  • TPM — Tokens Per Minute
  • Daily token cap
  • Real-time usage dashboard with quota bars
⚡

SSE Streaming

All chat responses stream token-by-token via Server-Sent Events, showing the provider and model serving the request, activity steps in real time, and debate rounds as they complete.

📦

Response Caching

Identical prompts are cached for 24 hours via SHA-256 hashing — avoids redundant API calls and reduces cost for repeated queries. Configurable toggle in Settings.

✂️

Context Optimization

Two optimization layers keep context within model windows:

  • Compression — Normalizes whitespace and strips system prompt boilerplate
  • History Trimming — Removes least important messages when approaching context limit
🎨

Image Generation

Gemini's Imagen model is available for text-to-image generation with inline display in chat.

📁

Projects

Organize conversations into named projects. Create, rename, and delete projects with a filterable sidebar.
