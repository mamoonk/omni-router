Omni-Router

Multi-provider AI chat assistant with smart routing, quota management, debate mode, and a code agent — all running locally as an Electron desktop app.

View on GitHub Install Now

Features

Smart Routing

Routes each message to the best provider based on intent, quota, speed, and failure history.

20+ Providers

Gemini, OpenAI, Anthropic, Groq, Mistral, Cohere, DeepSeek, and many more.

Debate Mode

Two AI models discuss and refine each other's answers for higher quality results.

Code Agent

Reads and modifies source code directly — works on the app itself or any external project.

Quota Management

Per-provider rate limits (RPM, TPM, daily caps) with a real-time usage dashboard.

SSE Streaming

Token-by-token streaming with live activity steps showing routing decisions.

Install

Prerequisites: Node.js 18+, npm, and git.

curl -fsSL https://raw.githubusercontent.com/mamoonk/omni-router/master/install.sh | bash
iwr -Uri https://raw.githubusercontent.com/mamoonk/omni-router/master/install.ps1 | iex
# Clone the repo
git clone https://github.com/mamoonk/omni-router.git
cd omni-router

# Install dependencies & build
npm install
npm run build

# Run in dev mode
npm run dev

Run

npm run devElectron app with hot-reload
npm run build:web && npm run start:webStandalone web server
npm run build && npm run previewProduction preview
npm run distPackage distributable (DMG/AppImage/EXE)
